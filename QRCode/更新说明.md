# 🎉 版本更新说明 v2.0

## 📅 发布日期
2025年12月23日

## 🆕 重大更新：HTML数字名片功能

### 功能概述
全新推出"HTML名片/简历"生成模式，用户可以创建专业的响应式HTML个人主页，并通过QR码分享。告别枯燥的纯文本二维码，让你的数字名片更加出色！

### 核心亮点

#### 1. 信息全面（11个字段）
- 👤 基础信息：姓名、职位、公司
- 📞 联系方式：邮箱、电话、微信
- 🌐 在线主页：个人网站、GitHub
- 📝 个人展示：简介、技能标签、所在地

#### 2. 专业设计
- **视觉效果**：紫色渐变背景 + 白色卡片布局
- **动画效果**：淡入加载动画，提升用户体验
- **响应式**：完美适配手机、平板、电脑
- **可交互**：邮箱/网站/GitHub自动转为可点击链接

#### 3. 一站式流程
```
填写信息 → 下载HTML → 上传托管平台 → 获取URL → 生成QR码 → 分享使用
```

#### 4. 实时预览
- 应用内嵌HTML预览窗口
- 无需离开应用即可查看效果
- 所见即所得

#### 5. 托管指引
提供详细的平台选择建议：
- **推荐方案**：GitHub Gist（最简单，1分钟部署）
- 其他选项：GitHub Pages / Netlify / Vercel

### 技术特性

- **无依赖**：纯HTML + CSS，无需服务器环境
- **轻量级**：单文件不到5KB，加载迅速
- **SEO友好**：语义化标签，搜索引擎可索引
- **可定制**：用户可手动编辑HTML修改样式
