# 📄 HTML名片/简历功能使用指南

## ✨ 功能介绍

新增的"HTML名片/简历"功能可以将您的个人信息生成为一个精美的响应式HTML页面，然后通过QR码分享。扫码者将看到一个专业的个人名片网页，而不是普通的纯文字信息。

## 🎯 适用场景

- 📇 **数字名片**：在会议、活动中快速分享联系方式
- 👔 **求职简历**：将简历做成网页，方便HR扫码查看
- 👨‍💻 **个人品牌**：展示技能、作品集和社交媒体
- 🤝 **商务交流**：更专业的自我介绍方式

## 📝 使用步骤

### 步骤1：填写个人信息

在左侧边栏选择 **"HTML名片/简历"** 模式，然后填写：

- 👤 **姓名**（必填）
- 💼 **职位/头衔**
- 🏢 **公司/组织**
- 📧 **邮箱**（推荐填写）
- 📱 **电话**
- 💬 **微信号**
- 📍 **所在地**
- 🌐 **个人网站**
- 💻 **GitHub**
- 📝 **个人简介**
- 🎯 **技能标签**（支持逗号或换行分隔）

### 步骤2：预览HTML效果

在主界面底部的 **"👁️ 预览HTML效果"** 展开区域，可以实时查看生成的HTML名片样式。

### 步骤3：下载HTML文件

点击 **"⬇️ 下载 HTML 名片"** 按钮，保存为 `.html` 文件到本地。

### 步骤4：上传到托管平台

选择以下任一免费托管平台：

#### 推荐方案：GitHub Gist（最简单）

1. 访问 [gist.github.com](https://gist.github.com)
2. 将下载的HTML文件内容粘贴到编辑框
3. 文件名改为 `index.html`
4. 点击 **"Create public gist"**
5. 点击右上角 **"Raw"** 按钮
6. 复制浏览器地址栏的完整URL

**优点**：无需注册，永久免费，一分钟搞定

#### 其他方案

- **GitHub Pages**：支持自定义域名，更专业
- **Netlify Drop**：拖拽即可部署，零配置
- **Vercel**：专业托管，速度快

### 步骤5：生成二维码

1. 将获取的HTML链接粘贴到右侧 **"输入HTML页面链接"** 输入框
2. 预览二维码效果
3. 点击 **"⬇️ 下载二维码"** 保存

### 步骤6：分享使用

- 将二维码印刷到名片、海报、简历上
- 发送给朋友或客户
- 对方扫码即可看到精美的HTML名片页面！

## 💡 使用技巧

1. **简洁为美**：不必填写所有字段，只填必要信息即可
2. **技能分类**：技能标签建议6-10个为宜，突出核心竞争力
3. **链接测试**：上传后先自己扫码测试，确认链接正确
4. **定期更新**：可以随时修改HTML重新上传，QR码链接不变

## 🔧 常见问题

**Q: GitHub Gist 的链接是什么样的？**  
A: 类似 `https://gist.githubusercontent.com/username/xxxxx/raw/xxxxx/index.html`

**Q: 可以自定义HTML样式吗？**  
A: 可以！下载HTML后用文本编辑器打开，修改 `<style>` 标签内的CSS代码

**Q: HTML会过期吗？**  
A: GitHub Gist 和 GitHub Pages 是永久免费的，只要不删除就不会过期

**Q: 可以添加照片吗？**  
A: 当前版本暂不支持，可以手动编辑HTML添加 `<img>` 标签

**Q: 如何更改背景颜色？**  
A: 编辑HTML中 `body` 的 `background` 属性，例如改为红色渐变：
```css
background: linear-gradient(135deg, #ff6b6b 0%, #ee5a6f 100%);
```

## 🚀 高级玩法

1. **多语言版本**：创建多个HTML（中英文），用不同QR码分享
2. **A/B测试**：测试不同简介和配色的转化效果
3. **数据追踪**：使用短链服务（如 bit.ly）包装链接，追踪扫码数据
