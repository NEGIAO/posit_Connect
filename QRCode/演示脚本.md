
**操作**：

### Part 5: 下载HTML（15秒）
```
画面：点击"下载 HTML 名片"按钮
旁白：接下来，我们下载这个HTML文件
```

**操作**：
- 点击下载按钮
- 展示浏览器下载通知
- （可选）在本地浏览器打开HTML预览

---

### Part 6: 上传到GitHub Gist（40秒）
```
画面：切换到 GitHub Gist 网站
旁白：现在我们需要把这个HTML托管到网上，
     我推荐使用GitHub Gist，完全免费且永久有效
```

**操作步骤**：
1. 打开 https://gist.github.com
2. 粘贴HTML内容到编辑框
3. 文件名改为 `index.html`
4. 点击 "Create public gist"
5. **重要**：点击右上角 "Raw" 按钮
6. 复制地址栏URL

**展示URL示例**：
```
https://gist.githubusercontent.com/username/abc123.../raw/.../index.html
```

---

### Part 7: 生成QR码（30秒）
```
画面：切换回 Streamlit 应用
旁白：有了托管链接，我们就可以生成QR码了
```

**操作**：
1. 回到应用右侧 "生成二维码" 区域
2. 粘贴刚才复制的URL
3. 等待QR码自动生成
4. 展示预览效果

**可选**：
- 调整样式（切换到"商务蓝"）
- 添加中心图标

---

### Part 8: 下载与测试（20秒）
```
画面：下载QR码并用手机扫描
旁白：最后，我们下载这个QR码，用手机扫一扫
```

1. 点击 "下载二维码" 按钮
2. （拿出手机）扫描屏幕上的QR码
3. **展示手机画面**：
   - 浏览器打开HTML名片
   - 展示流畅的响应式效果
   - 点击邮箱链接（调起邮件应用）
   - 滚动查看完整信息
